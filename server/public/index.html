Calculator #1 Description
The user enters 2 integer values in the input fields provided. An operator can be selected by clicking on one of the four options (addition, subtraction, multiplication, division). Once the equation is written, clicking the equals sign will send the equation to the server where the math logic is performed. The equation and answer is then sent back to the DOM where it is appended beneath the "Calculator #1 Equation History" header.
The "C" button clears out any values in the integer input fields.

Calculator #2 Description
The user enters an equation using the keypad. The equation can be seen in the calculator's display as it is being entered. Once the equation is written, clicking on the equals sign will send the equation to the server where the math logic is performed. The equation and answer is then sent back to the DOM. The answer is then displayed on the DOM underneath the "Answer" header. A history of the equations are stored on the DOM underneath the "Calculator #2 Equation History Header" however they don't include the answer.
If a user would like to view an answer to a previously performed equation, they can simply click on that equation in the equation history. That equation is then sent back to the server where the math logic is performed again and the answer is sent back to the DOM where it will be shown in the calculator's display as well as in the Answer section.
The "clear display" button will clear out all data in the calculator's display. The "C" button will remove the most recent equation from the Equation History section as well as remove the answer from the Answer section.
